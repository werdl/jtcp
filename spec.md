## JTCP - for moving data more efficiently

### Format
```json
{
    "jtcp": {
        "version": "0.1.0"
    },

    "host": {
        "os": {
            "host-triple": "x86_64-unknown-linux-gnu",
            "distro": "debian-12"
        },
        "net": {
            "ipv4": "1.2.3.4",
            "ipv6": "2001:0db8:85a3:0000:0000:0000:0000:0000"
        }
    }
}
```
